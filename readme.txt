Usage Instructions
Run the Program:

python analyze_traceroute.py <pcap_file>
Replace <pcap_file> with the path to your PCAP file.

Batch Analysis (Group Files): Uncomment the relevant section in the main block to process Group 1 or Group 2 trace files.

Output Includes:
Source and destination IPs.
Intermediate router IPs.
RTT statistics (average and standard deviation).
Probe counts per TTL.
